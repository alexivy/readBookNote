# 常用命令  


***ctrl + L***   
清屏  

***top***   
整机状态，cpu内存占用 load average 3数平均大于60%说明系统负担压力够 按1看cpu具体  

***uptime***   
top命令精简版  

***vmstat -n 2 3***   
cpu状态，2秒采样1次，总共采样3次，r代表运行线程，b代表阻塞线程  

***mpstat -P ALL 2***   
cpu核信息，2秒采样一次  

***pidstat -u 1 -p 进程编号***   
进程使用cpu的信息  

***pidstat -p 进程号 -r 采样率***   
查看进程的占用内存百分比  

***pidstat -d***   
查看进程的磁盘应用状况  

***free***   
查看内存，-g单位是GB，-m单位MB，7成充足2成缺少  

***df -h***   
查看硬盘剩余空间（占用情况）  

***iostat -xdk 2 3***   
util表示每秒有多少时间用于IO操作  

***ifstat***   
查看网络状况
